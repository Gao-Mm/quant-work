#Portfolio Analytics
This projects involves data cleaning of a portfolio of stocks, a line graph and calculating the return of portfolio given hypothetical weights.

## Credits 
The dataset was from kaggle from Mohamed Mahmoud Ali's notebook: https://www.kaggle.com/datasets/thesnak/stock-market-analysis.

## Requirements
The notebook was tested on Google Colab. The following Python packages are required:
pandas
matplotlib
numpy 

## How to run
1. Open notebook in google colab:
2. https://colab.research.google.com/github/Gao-Mm/quant-work/blob/main/portfolio_analytics.ipynb
3. Include required packages included already in google collab.
